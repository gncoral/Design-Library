### 🤖 新增自动化：数英精选每日抓取

新脚本 `fetch-digitaling-auto.js`，每天10:05自动运行，完整流程：
1. 调数英两个API（编辑精选 + 本周热门）
2. 合并去重，按收藏数×2 + 点赞数综合排序
3. 优先选14天内发布的文章，取 top 1
4. 自动下载封面图，上传至 GitHub 自托管
5. 写入 digitaling.json，推送 GitHub

今日首跑入库：**「淘宝第一文案「步履不停」，又出神作了」**（收藏62 / 点赞30）
